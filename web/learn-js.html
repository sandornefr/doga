Az alert() egy beépített JavaScript függvény, amely egy felugró üzenetablakot jelenít meg.

Szintaxis: alert("szöveg");

Hasonló beépített függvények:
  • alert("szöveg") – üzenet megjelenítése
  • confirm("kérdés") – Igen/Nem választás (igaz/hamis visszatérési érték)
  • prompt("kérdés") – szöveges bevitel a felhasználótól
